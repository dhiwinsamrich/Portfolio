# Archive Components

This folder contains old/unused components that may be reused in the future.

## Components

### hero-parallax.tsx
The original parallax hero section with scrolling product cards. This was replaced by the new letter-based hero section (hero-03.tsx).

**Usage:**
```tsx
import { HeroParallax } from '../components/archive/hero-parallax';

<HeroParallax products={products} />
```

**Note:** Make sure to import the products array when using this component.
